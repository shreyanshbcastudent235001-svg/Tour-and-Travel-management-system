# Tour-and-Travel-management-system
The Tour and Travel Management System is a web-based application designed to simplify and enhance the process of planning, booking, and managing travel services. This project provides users with an intuitive interface where they can explore various travel destinations, view tour packages, and make bookings. 
Tour & Travel - Simple PHP Project
---------------------------------
What's included:
- Basic PHP pages (index, signup, login, package listing, booking flow)
- Admin panel (login, dashboard, add/manage packages, manage bookings)
- include/db.php for DB connection
- sql/database.sql to create database and tables (see below)

Default DB settings (edit include/db.php):
host: localhost
user: root
pass: (empty)
database: tour_travel_db

Admin credentials (from sql/database.sql):
username: admin
password: admin   (stored as MD5 in SQL; admin login validates with MD5 for simplicity)

Notes:
- After importing sql/database.sql into MySQL, update DB credentials in include/db.php.
- This project is a starter template. Sanitize inputs and secure accordingly before production.
